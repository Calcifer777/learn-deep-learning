PageRank

Algorithm​

Main idea: the importance of a node vvv is proportional to the importance of its neighbors.

In particular, it is proportional to the importance of its neighbors with a out-link towards vvv.

Flow formulation​

rj=∑i→jridir_{j} = \sum_{i \rightarrow j}{\frac{r_i}{d_i}}rj​=i→j∑​di​ri​​

where ridi\frac{r_i}{d_i}di​ri​​ is the importance of node iii normalized by its out-degree

Matrix formulation​

We can define a stochastic adjacency matrix M∈R∣G∣×∣G∣M \in \mathbb{R}^{|G| \times |G|}M∈R∣G∣×∣G∣, where each column iii contains to the out-links of node iii to all the other nodes in the graph. The non-zero elements of column iii are all 1di\frac{1}{d_i}di​1​, with did_idi​ the out-degree of node iii.

We can also define a vector r∈R∣G∣r \in \mathbb{R}^{|G|}r∈R∣G∣ which encodes the importance of each node, and normalize its values so that ∑r=1\sum{r} = 1∑r=1.

Then, the flow equation can be re-written as:

r=Mrr = M rr=Mr

Random walk formulation​

The algorithm can be also viewed as the result of applying an infinite uniform random walk from any node.

The dynamic equation that describes the probability of the walker to be at a certain node at a given step is:

p(t+1)=Mp(t)p(t+1) = M p(t)p(t+1)=Mp(t)

It this sense, the importance vector rrr in the matrix formulation can also be interpreted as the stationary distribution for the random walk.

Eigenvector formulation​

The matrix formulation can be rewritten as:

1⋅r=Mr1 \cdot r = M r1⋅r=Mr

So the rank vector rrr is an eigenvector of the stochastic adjacency matrix MMM, corresponding to the eigenvalue with value 111.

Solving PageRank​

Issues with the basic algorithm:

  • dead ends: cause importance to 'leak out'
  • spider traps: absorb all the importance

Solution: allow the walker to teleport to a random node in the graph with probability β\betaβ; in that case, the average importance of the destination node would be 1N\frac{1}{N}N1​, as the elements of rrr sum to 111.

The complete algorithm in the flow formulation then becomes:

rj=(1−β)∑j→iridi+β1Nr_j = (1-\beta)\sum_{j \rightarrow i}{\frac{r_i}{d_i}} + \beta \frac{1}{N}rj​=(1−β)j→i∑​di​ri​​+βN1​

In the matrix formulation, the matrix used for the recursion becomes

G=βM+(1−β)[1N]N×N matrix with all entries 1NG = \beta M + (1 - \beta)\underset{N \times N \text{ matrix with all entries }\frac{1}{N}}{\left[ \frac{1}{N} \right]}G=βM+(1−β)N×N matrix with all entries N1​[N1​]​

Variants​

Personalized PageRank​

Modifies the original PageRank algorithm by allowing the node to teleport only to a subset of nodes in the graph.

Can be used to rank the proximity of nodes in the graph with respoct to a subset of query nodes.

PageRank with restarts​

The set of possible nodes to which the walker can teleport is restricted to a single node.

The set of query nodes effectively becomes a singleton.
